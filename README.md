# gradle-magicdraw
Gradle plugin to develop plugins for MagicDraw

## Motivation

## Usage
Simple use the `plugins` block to apply the plugin to your project:

    plugins {
        id '' version ''
    }

The plugin will not apply any other plugin automatically.
